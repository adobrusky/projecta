Media Query Break Points

The break points for ProjectA are at 435px represented as S, 620px represented as MS, 768px represented as M, and 1024px represented as L.


Dynamic Variables

We have a few dynamic variables in ProjectA. These are varaibles you can change in an external javascript sheet to change some of the default properties of ProjectA. If you can't understand my descriptions you can just play with them and you will understand what they do.


  • carouselInterval: this is the interval at which a carousel changes slides.

    Default: 1000

  • navItemWidth: this is factor in which you change the percentage of the width of the navbar items when not collapsed.

    Default: 80

  • navDuration: this is the duration of all of the collapsed navbar transitions

    Default: 600

  • navShrink: this is the shrink factor of all of the navbar items when expanded

    Default: 2.5


Atom Snippets

We have immensily sped up the process of working with ProjectA if you are using Atom as a text editor. At the github master branch for ProjectA we will have access to the downloads for our snippets. A guide on how to install will be up shortly. Once the snippets are installed, simply type the keyword and press tab.


  • projecta: generates a full html structure while referencing projecta and jquery along with it
  • nav: generates a projecta navbar
  • content: generates the content section of the webpage
  • dropdown: generates a navbar dropdown in place of a normal list item
  • sub: generates a dropdown sub-menu in place of a list item inside of a dropdown
  • vars: generates all of the dynamic variables. use this in a javascript file
  • carousel: generates a carousel
  • row-(size)-(columns): generate the outline for our gridsystem. In size put the point you want it to collapse at. In columns put how many columns you want(2-4). Ex: row-m-2
